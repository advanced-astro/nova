## Usage

Import both the `Carousel` and `CarouselSlide` components where you want to use them.

```jsx
import Carousel from '@components/Carousel.astro';
import CarouselSlide from '@components/CarouselSlide.astro';
```

You can then use the `Carousel` component with `CarouselSlide` components as children.

```astro
<Carousel withArrows withDots>
	<CarouselSlide>...</CarouselSlide>
	<CarouselSlide>...</CarouselSlide>
</Carousel>
```

The carousel accepts `withArrows` or `withDots` props, both are optional.

### Scooch Options

The Carousel uses Scooch to power the carousel. You can pass options to Scooch using the `options` prop and passing an object of Scooch options.

```jsx
<Carousel options={{ autoplay: false, keyboardControls: false }}></Carousel>
```
